Demostración de Tipos de <input> HTML

Exploración de diferentes tipos de input con estilos definidos en styles.css.

Tipos de Entrada (Elemento <input>)

type="text"

Campo de texto de una sola línea.

type="password"

Campo de texto donde los caracteres se ocultan.

type="submit"

Botón para enviar un formulario.

type="reset"

Botón para limpiar los campos de un formulario.

type="radio"

Selección única dentro de un grupo.

type="checkbox"

Selección múltiple.

type="button"

Botón genérico (acción definida con JS).

type="color"

Selector de color.

type="date"

Selector de fecha.

type="datetime-local"

Selector de fecha y hora local.

type="email"

Campo para correo electrónico (con validación básica).

type="file"

Selector de archivos.

type="hidden"

Campo oculto (no visible).

(Este input no es visible)

type="image"

Botón de envío con imagen.

type="month"

Selector de mes y año.

type="number"

Campo numérico con controles opcionales.

type="range"

Control deslizante para un rango numérico.

type="search"

Campo de texto para búsquedas.

type="tel"

Campo para número de teléfono.

type="time"

Selector de hora.

type="url"

Campo para URL (con validación básica).

type="week"

Selector de semana y año.
